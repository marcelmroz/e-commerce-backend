- POST /api/customers
- POST /api/carts
- POST /api/customers/login

- GET /api/carts/id
- GET /api/products
- GET /api/customers
- GET /api/carts
- PUT /api/products/id
- PUT /api/customer/id
- PUT /api/carts/id
- DELETE /api/products/id
- DELETE /api/customers/id
